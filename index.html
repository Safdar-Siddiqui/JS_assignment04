05 legal names for variables in JS.

  1. Character i.e. var x;
  2. Character String i.e. var safdar;
  3. Having numbers in variable name i.e. var str1;
  4. Having dollar sign in variable name i.e. var $safdar;
  5. Having underscore in variable name i.e. _safdar or safdar_siddiqui

05 illegal legal names for variables in JS.

  1. Cannot only be number i.e. var 123;
  2. Cannot start with number i.e. var 123safdar;
  3. Cannot have space i.e. var safdar siddiqui;
  4. Cannot only be Javascript keywords i.e. var let; but may contain keywords i.e. var letDesign;
  5. Cannot have Special Characters except $ and _ i.e. @,#, %, ^ etc.

Rules for naming JS Variables

  1. Variable names can only contain numbers, $ and _. For example: $my_1stVariable
  2. Variable must begin with a letter, $ or _. For example: $name, _name or name
  3. Variable names are Case Sensitive
  4. Variable names should not be JS Keywords
